# رستوران ال‌من

یک وب‌سایت رستوران مدرن با React و Tailwind CSS

## ویژگی‌ها
- طراحی واکنش‌گرا برای تمامی دستگاه‌ها
- بهینه شده برای موبایل
- پشتیبانی از زبان فارسی و راست‌به‌چپ
- فرم رزرو آنلاین
- گالری تصاویر غذاها
- منوی تعاملی

## فناوری‌های استفاده شده
- React
- Vite
- Tailwind CSS
- JavaScript ES6+

## نحوه اجرا
1. نصب وابستگی‌ها:
   ```
   npm install
   ```

2. اجرای سرور توسعه:
   ```
   npm run dev
   ```

3. ساخت فایل‌های نهایی:
   ```
   npm run build
   ```

4. پیش‌نمایش نسخه تولید:
   ```
   npm run preview
   ```

## مستقر کردن روی GitHub Pages
1. ایجاد repository در GitHub
2. اضافه کردن remote:
   ```
   git remote add origin https://github.com/MrAbol777/portfolio-1.git
   ```
3. پوش کردن کدها:
   ```
   git add .
   git commit -m "Initial commit"
   git push -u origin main
   ```
4. تنظیم GitHub Pages در Settings > Pages
